Sample codes for nim and GTK 3.0
=========================================================
nim samples of GTK+ 3.0.


Prerequiresites/ Dependency
--------------------------
This samples were tested in

- Debian 11.5
- WSL2 on Windows 11
- nim 1.4.2
- libgtk-3-dev package



Build
--------------------------

```shell
$ nim c simple.nim
$ ./simple
```
